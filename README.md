# Tarefas LocalStorage

Aplicacao de tarefas com cadastro, conclusao, filtros e persistencia no navegador.

## Objetivo

Praticar CRUD no front-end, manipulacao de estado, eventos, filtros e armazenamento local.

## Funcionalidades

- Criar tarefas com prioridade.
- Marcar tarefas como concluidas.
- Remover tarefas.
- Filtrar por todas, pendentes e concluidas.
- Contadores de tarefas.
- Persistencia com `localStorage`.

## Tecnologias

- HTML
- CSS
- JavaScript

## Como rodar

Abra o arquivo `index.html` no navegador.
